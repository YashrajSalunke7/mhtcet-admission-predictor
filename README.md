# 🎓 MHT-CET Admission Prediction System

An AI-powered college recommendation engine built with Python and Streamlit.

## Features
- Predicts admission chances using RandomForest ML model
- CAP-style Dream 🔥 / Target 🎯 / Safe ✅ college classification
- Real MHT-CET 2024 cutoff data across 349 colleges, 101 branches
- Synced percentile input (type or slide)
- Download results as CSV

## Tech Stack
- Python, Streamlit, Scikit-learn, Pandas, Matplotlib

## Run Locally
pip install -r requirements.txt
streamlit run app.py
